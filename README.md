# 📝 FastAPI To-Do List

Сучасний веб-додаток для управління списком завдань, розробленний на **FastAPI** з аутентифікацією користувачів та постійним збереженням даних.

🔗 **Живий проект:** https://todo-app-l42x.onrender.com/home

---

## 📚 Про проект

**FastAPI To-Do List** — це полнофункціональний додаток для управління персональними завданнями. 

Кожен користувач має:
- ✅ Особистий список завдань
- 🔐 Безпечну аутентифікацію
- 📅 Історію видалених завдань

---

## 🎯 Функціонал

| Функція | Опис |
|---------|------|
| **Реєстрація & Логін** | Безпечна аутентифікація з хешуванням паролів (bcrypt) |
| **Додавання завдань** | Швидке створення нових задач |
| **Позначення завдань** | Відмітити завдання як виконане/невиконане |
| **Видалення завдань** | Мякке видалення (переміщення в історію) |
| **Історія** | Переглядайте видалені завдання |
| **Персональне середовище** | Кожен користувач бачить тільки свої завдання |
| **Форматування часу** | Час ство��ення в зручному форматі (DD.MM.YYYY HH:MM) |

### **Чому така архітектура?**

✅ **FastAPI** — мінімум коду, максимум продуктивності

✅ **SQLAlchemy** — безпека від SQL-injection, простота роботи з БД

✅ **PostgreSQL** — надійна, масштабована база даних

✅ **JWT/Cookies** — простий і безпечний способ управління сесіями

✅ **Jinja2** — динамічне рендерування HTML на сервері

## 📥 Установка та запуск

### **Вимоги:**
- Python 3.11+
- PostgreSQL (для прода) / SQLite (для розробки)
- Git

### **Крок 1: Клонування репозиторію**

```bash
git clone https://github.com/ZelenaZelenaZapka/FastApi_ToFoList.git
cd FastApi_ToFoList
```
### **Крок 2: Налаштування окремого середовища**
```
  # Автоматичне налаштування (Linux/Mac)
  bash setup.sh
  
  # Або вручну:
  python -m venv myenv
  source myenv/bin/activate  # На Windows: myenv\Scripts\activate
```
### **Крок 3: Установка залежностей**
```
pip install -r requirements.txt
```
### **Крок 4: Налаштування змінних середовища**
```
# Скопіюй файл
cp env.example .env

# Відредагуй .env:
DATABASE_URL=postgresql://user:password@localhost/todo_db
```
### **Крок 5: Запуск сервера**
```
# Локально
uvicorn main:app --reload або fastapi dev

# Або з bash скриптом
bash run_tests.sh або pytest tests/ -v # Запускає тести
```
Що тестується:

    ✅ Аутентифікація (реєстрація, логін, вихід)
    ✅ Управління завданнями (додавання, видалення, позначення)
    ✅ Контроль доступу (користувач не може видалити чужі завдання)
    ✅ Форматування часу
    ✅ Валідація даних

🔒 Безпека

    ✅ Паролі хешуються з Bcrypt (не можна відновити)
    ✅ Cookies з httponly флагом (захист від XSS)
    ✅ SQL Injection захист (SQLAlchemy)
    ✅ CSRF токени в формах
    ✅ Тести перевіряють ізоляцію користувачів

📊 Дані, що зберігаються

Users:
id | email | hashed_password | created_at

Tasks:
id | title | completed | deleted | deleted_at | created_at | user_id

Всі дати зберігаються в UTC для максимальної надійності.

### **🚀 Deploy на Render.com**

    Підключи GitHub репозиторій до Render
    Налаштуй змінні середовища (DATABASE_URL)
    GitHub Actions автоматично буде запускати тести
    Якщо тести пройдуть → Render деплоїть нову версію

Все відбувається автоматично при git push 🎉
